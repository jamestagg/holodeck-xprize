# Holodeck XPRIZE

Holodeck is a Gemini-powered world engine for experiential learning.

The project turns history, science, literature, and imagination into interactive places that learners can enter, explore, and learn from. The first commercial product is **Holodeck Season 1: Time Travel**, a paid founder-access campaign for explorable historical worlds.

## XPRIZE Category

Education & Human Potential

## Initial Worlds

- Apollo 11: Mission Control
- Alexandria: The Last Library
- Pompeii: The Final Day
- Bletchley Park: Break the Code
- Florence: Workshop of the Renaissance

## Gemini Use

Gemini is planned as the core reasoning layer for:

- source-grounded world briefs
- character motivations and bounded knowledge
- adaptive dialogue
- mission generation
- tutoring and hints
- session summaries
- learner assessment
- campaign and customer operations

## Business Model

Holodeck will monetize through a Kickstarter-style founder campaign:

- Explorer Pass
- Founder Pass
- Family Pass
- Educator Pack
- Patron World
- Custom Pilot
